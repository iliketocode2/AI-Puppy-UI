AI Puppy Lesson 5: Heel

Code your puppy to follow your hand with the distance sensor using one of the machine learning methods below!

Machine Learning Type

1D K-Nearest Neighbor, Linear Regression

Tools Used

Distance sensor, motors, & starter code

Example Ideas

Build a puppy that can be petted!

Tips & Troubleshooting

Data collection for this lesson requires an extra step! Both of the following functions must be used:

ai.get_data
ai.add_data

Helpful functions for play mode:

ai.linreg_prediction
ai.puppy_drive

If your puppy seems to be moving in the opposite direction you want it to, switch the left and right motor ports!
