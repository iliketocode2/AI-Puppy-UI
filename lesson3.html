AI Puppy Lesson 3: Actions

Code your puppy to complete at least 3 actions based on what a sensor reads.

Machine Learning Type

1D K-Nearest Neighbor

Tools Used

One sensor (any), motors, & starter code

Example Ideas

Build a puppy that can be petted!

Tips & Troubleshooting

Start with coding 2 actions first.

Check sensors & motors are defined correctly.

Check syntax & typos.

Look at the line the error is being called at.

Make sure the left, center, and right hub buttons are easily accessible.
